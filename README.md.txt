DevMentor IA 🤖🐍

O DevMentor IA é um mentor virtual desenvolvido para auxiliar estudantes e entusiastas de tecnologia com tópicos de TI. Ele fornece explicações de conceitos, roteiros de estudo e exemplos práticos de código de forma interativa e rápida.

Este projeto foi desenvolvido como parte do desafio do Módulo 1 do curso Fundamentos de Linguagem Python - Do Básico a Aplicações de IA da Data Science Academy.

🚀 Tecnologias Utilizadas

•
Python: Linguagem base do projeto.

•
Streamlit: Framework utilizado para criar a interface web interativa.

•
Groq Cloud: Infraestrutura de processamento de LLM (Large Language Model) para respostas ultra-rápidas.

🛠️ Como rodar o projeto localmente

1.
Clone o repositório:

Bash


git clone https://github.com/SEU_USUARIO/devmentor-ia.git





2.
Instale as dependências:

Bash


pip install streamlit groq





3.
Execute a aplicação:

Bash


streamlit run app.py



(Certifique-se de ter sua chave de API da Groq em mãos! )
